We developed OOMOL Studio based on VSCode, a workflow solution that supports Windows and macOS. ## Pain points solved - Cumbersome local environment configuration, painful cross-platform development - Want to use GPU acceleration, but the container environment is not easy to configure - Inconsistent environment during team collaboration, debugging is very painful - Want a unified workspace to manage various projects - Just want to write code, don't want to bother with environment and UI ## Core features - **Cross-platform support**: - Windows - macOS - **Virtualization capabilities**: - Podman-based containerization solution - GPU support - Simultaneous support for Mac's Apple and Intel chips - **AI**: - Built-in large model API - Support for mainstream large models - Support for AI code completion - Free 1M Token per month - **Community-driven ecosystem**: - One-click sharing of workflows (Flows) - Ready-to-use script library (Blocks) - Drag and drop to reuse excellent solutions from others - Developer community - **Modern technology stack**: - Front-end: Deep customization based on VSCode - Back-end: Node.js + Golang + Rust + C ## Open source Although OOMOL Studio itself is not open source, we deeply embrace open source: 1. Active community contributions: - Contribute code to well-known projects such as [VSCode]( https://github.com/microsoft/vscode), [podman]( https://github.com/containers/podman), [urfave/cli]( https://github.com/urfave/cli), [cnpm]( https://github.com/cnpm/cnpmcore), [vfkit]( https://github.com/crc-org/vfkit) - Sponsor and support the development of excellent open source projects such as [vz]( https://github.com/code-Hex/vz) - Continue to participate in open source community construction ## Practical scenarios - AI model development (supports GPU acceleration) - Cross-platform project development - Data processing and analysis - Team collaborative development - DevOps workflow - Quickly reuse excellent community practices - Share your workflow with others ## Screenshots ![]( https://i.v2ex.co/SY2DLSZW.png) ![]( https://i.v2ex.co/5Y2BGG3s.png) ![]( https://i.v2ex.co/i8fJxeRX.png) ## Benefits To thank V friends for their support, we have prepared 20 Xiaomi insulated cups and will draw 20 lucky V friends from the comments section (deadline February 27). > Lottery program: https://hub.oomol.com/package/v2ex-lottery/0.0.1 ## Finally If you are interested in OOMOL Studio: 1.
